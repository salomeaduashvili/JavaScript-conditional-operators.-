# JavaScript Practice Tasks

This project contains **11 JavaScript exercises** covering numbers, strings, and basic conditional operators.
Each task is wrapped in a function and can be run separately.  
All user inputs are taken using **prompt-sync**.

ეს პროექტი შეიცავს **11 JavaScript ამოცანას**, რომლებიც მოიცავს რიცხვებს, სტრიქონებს და საბაზისო პირობით ოპერატორებს.  
ყოველი ამოცანა ფუნქციაშია ჩასმული და გამოყოფილად იძახება.  
ყველა ინფუთი მიიღება **prompt-sync**–ის გამოყენებით.

---

## ✨ Functionality / ფუნქციონალი
- Check if a number is even or odd / შეამოწმოს რიცხვი ლუწია თუ კენტი  
- Check if a number is positive, negative, or zero / შეამოწმოს რიცხვი დადებითია, უარყოფითი თუ ნული  
- Check divisibility / შეამოწმოს იყოფა თუ არა რიცხვი სხვა რიცხვზე  
- Check if a string is empty / შეამოწმოს სტრიქონი ცარიელია თუ არა  
- Check if a string contains a word / შეამოწოს, შეიცავს თუ არა სტრიქონი კონკრეტულ სიტყვას  
- Find minimum and maximum of two numbers / იპოვოს მინიმალური და მაქსიმალური ორი რიცხვიდან  
- Determine adult or minor / დაადგინოს არის თუ არა მომხმარებელი 18 წლის ან უფროსი თუ არასრულწლოვანი  
- Assign grades based on score / გამოიტანოს შეფასება ქულის მიხედვით (if/else და switch)  
- Print message based on grade / დაბეჭდოს შეტყობინება შეფასების მიხედვით  (if/else და switch)
- Determine number of days in a month / გამოიტანოს დღეების რაოდენობა თვეში  (if/else და switch)

---

## 🚀 Run / გაშვება

1. Install Node.js  
2. Install **prompt-sync**:
```bash
npm install prompt-sync
```
3. Run the script:
```bash
node script.js
```
